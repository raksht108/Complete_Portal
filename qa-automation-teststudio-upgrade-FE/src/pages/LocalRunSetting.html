Instruction

Please follow the instructions to execute test cases locally:

  1. Verify the JAR downloaded through the UI is placed under this folder: "C:\Automation\CEB_Final.jar".
  2. Verify config and driver files folders are placed under this folder: "C:\Automation".
  3. Check that the config folder contains the config.properties file, and the driver files folder has InternetExplorer.exe, Chrome.exe, and Firefox.exe files.
  4. If the above files and folders are found respectively, double-click the BAT file that has been downloaded to your computer so that execution will start on your machine.

For screenshots, videos, and logs:

  • Screenshots will be found in "C:\Automation\AutomatioNG_Reports\screenshot\browserName\TestRunID".
  • Videos will be found in "C:\Automation\AutomatioNG_Reports\Videos\TestRunID".
  • Logs will be found in "C:\Automation\LogFiles\TestRunID_TimeStamp".
Instruction

Please follow the instructions for installing the Chrome extension:

  1. Open "chrome://extensions/".
  2. Select the “Developer Mode” checkbox.
  3. Unzip the file.
  4. Click on the "Load unpacked extension" button.
  5. Select the folder, then click "OK".
  6. Ensure the CEB logo is showing in the top-right corner.

Please follow the instructions for inspecting objects and creating test cases:

  1. Go to your application page.
  2. Right-click on any object and select "Inspect" (or press F12).
  3. Click on the CEB Inspector tab in the developer tools.
  4. Click on the red "On" button to change it to blue (located in the top-right corner).
  5. Provide a page name.
  6. To create a test case, click the "+" sign in the top-right corner.
  7. Enter your test case name and description in the green bar by selecting the radio button.
